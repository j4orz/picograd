# picograd
*a bridge from [micrograd](https://github.com/karpathy/micrograd) to [tinygrad](https://github.com/tinygrad/tinygrad)* 

The hackable machine learning framework for the SITP course with
[textbook](https://j4orz.ai/sitp/) and [lectures]().
Follow along line by line in order to build your own distributed parallel
compiler capable of training and inferencing
[nanogpt](https://github.com/karpathy/nanoGPT) and [nanochat](https://github.com/karpathy/nanochat/).
Lectures, textbook, and codebase are created by the
[Singularity Systems Research Group](https://singularitysystems.bearblog.dev/)
and will always remain free.
