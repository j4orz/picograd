# picograd
*a teaching deep learning framework: the bridge from [micrograd](https://github.com/karpathy/micrograd) to [tinygrad](https://github.com/tinygrad/tinygrad)

The pedagogical deep learning framework made for the [SITP](https://j4orz.ai/sitp/) lectures and textbook.
Follow along and build your own deep learning framework line by line from scratch,
to run training and inference for [nanogpt](https://github.com/karpathy/nanoGPT) and [nanochat](https://github.com/karpathy/nanochat/).
