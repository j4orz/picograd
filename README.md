# picograd
*a teaching deep learning framework: the bridge from [micrograd](https://github.com/karpathy/micrograd) to [tinygrad](https://github.com/tinygrad/tinygrad)* 

The hackable pedagogical deep learning framework made for the [Machine Learning Systems: A Programmer's Perspective](https://j4orz.ai/mlsysapp/) textbook.
Follow along with the [lectures]() line by line, in order to build your own distributed parallel
compiler capable of running training and inference for [nanogpt](https://github.com/karpathy/nanoGPT) and [nanochat](https://github.com/karpathy/nanochat/).
